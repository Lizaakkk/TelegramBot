# TelegramBot
Telegram Bot: Конвертер валют
Описание:
Telegram-бот для конвертации валют с использованием API Центрального Банка РФ. Бот позволяет узнать цену определённого количества одной валюты в другой валюте.

Функционал:
Конвертация валют по курсу ЦБ РФ

Поддержка основных мировых валют (EUR, USD, RUB, CNY, GBP, JPY, CHF)

Обработка ошибок с информативными сообщениями

Структура:
bot.py - Основной файл бота, 
extensions.py - Классы для работы с валютами и API, 
config.py - токен
